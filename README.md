**Практика 2 (вопросы):**

в приложение?
Ответ: нет, не будет, т.к. есть обработка состояний onStop, onStart и onResume.

возврата в приложение?
Ответ: нет, не изменится.

3. Изменится ли значение поля «EditText» после нажатия на кнопку «Back» и
возврата в приложение?
Ответ: нет, не изменится.
